Individual User Profile & Community App – Functional Specification & User Flow 

Each user has a Client Profile containing:
- Profile photo
- Full name
- Area / neighbourhood
- Short bio (optional)
- Contact option (message)
- Joined date

Client Profile contains the following tabs:

1. Posts Tab
Features:
- Create posts (text & images)
- Post service requests
- Ask questions
- Share thoughts or updates
- Edit or delete own posts
- View engagement (likes & comments)

Use case examples:
- “Looking for a cleaner in my area”
- “Any recommendations for a handyman?”

2. My Favourites Tab
Features:
- Save service providers
- Save posts
- Save recommendations
- Remove items from favourites

3. Groups Tab
Features:
- Join public or private groups
- View group posts
- Leave groups anytime
- Group categories (Area-based, Services, Interests)

4. Invite People Tab
Features:
- Invite via phone contacts
- Invite via email
- Share invite link
- Track sent invites (optional)

Navigation:
Client Profile → Dashboard Icon → Main Dashboard
Clients can return to home feed from anywhere within profile.

SETTINGS TAB – USER FLOW

8. Legal & Policies

9. Share This App

10. Logout
